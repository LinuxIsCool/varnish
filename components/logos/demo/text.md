  en-US: Removing Organization Name Text
---

Depending on the context, you may or may not want to display the full
organization name text.

```jsx
import { Logos } from '@allenai/varnish';

const { AI2Logo } = Logos;

ReactDOM.render(
  <div>
    <AI2Logo includeText={false} size="lg" />
  </div>,
  mountNode,
);
```
