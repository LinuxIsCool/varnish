## en-US

Quotes and their authors use Volkhov Regular

```jsx
import { Typography } from '@allenai/varnish';

const { Quote, Author } = Typography;

ReactDOM.render(
  <div>
      <Quote>“Wit is the unexpected copulation of ideas.”</Quote>
      <br/>
      <Author>Patrick O&apos;Brian</Author>
  </div>,
  mountNode,
);
```
